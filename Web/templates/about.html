About Password Strength

Password Strength depends on variety of factors but 2 main factors are length and entropy. Entropy is considered as degree of randomness. It means that given a character from the password, is it possible to guess the next character. Higher the entropy, stronger the password. Length of the password also greatly determines the strength of the password. Longer the password, longer it will take to crack ift if we run a brute force algorihtm. Thus the entered passwords are checked for both of these features.

Our password generator module generates random string of uppercase letters, lowercase letters, digits and punctuation marks and feed them into the mentioned machine learning model. Only if the generated password is strong, we log that password into a text file so that we can generate Unique passwords each time. As the password generator module generates random string of uppercase letters, lowercase letters, digits and punctuation marks, the probability of finding the exact same string is 1/(76^15). Thus this ofefrs adequate entropy as well as length to make the password secure.
