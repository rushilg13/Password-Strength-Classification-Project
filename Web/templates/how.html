How our Website works?

Our Website uses Machine Learning and Natural Language Processing techniques to determine the strength of the passwords that user wants to check the stregth of. We convert the entered string into vectors and feed them to trained Machine Learning models and get the result as the strength of the entered password. We compared different machine learning algorithms and tested with hyperparameter tuning and compared their accuracies. We realised that, Decision Tree offers the best accuracy and hence we are using that algorihtm in the background. The Website uses pickled Machine learning model and unpacks them in our Python based Flask Backend. We process the string and send the data to the frontend.

Our password generator module generates random string of uppercase letters, lowercase letters, digits and punctuation marks and feed them into the mentioned machine learning model. Only if the generated password is strong, we log that password into a text file so that we can generate Unique passwords each time.
